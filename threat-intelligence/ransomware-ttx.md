# Ransomware Tabletop Exercise (TTX)

## Overview
This project outlines a simulated ransomware attack scenario designed to evaluate an organization’s incident response capabilities.

## Objectives
- Test incident response procedures
- Evaluate communication between teams
- Assess decision-making under pressure

## Scenario
- Initial access via phishing
- Exploitation of a zero-day vulnerability
- Rapid spread across network
- Data encryption and ransom demand

## Key Challenges
- Executive pressure during incident
- Public disclosure of breach
- Data leakage to external sources
- Overloaded help desk and IT teams

## Incident Response Phases
1. Preparation
2. Identification
3. Containment
4. Eradication
5. Recovery
6. Lessons Learned

## Outcome
This exercise improves organizational readiness, strengthens coordination, and highlights gaps in incident response planning.
